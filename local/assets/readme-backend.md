# Для бэкенда

### Модальные окна

Все модальные окна работают асинхронно и подтягиваются по запросу. Для открытия попапа на кнопку требуется два атрибута: href с ссылкой на модальное окно html-формата и data-popup="#id-модального-окна". Соответсвенно в начальной разметке старницы их быть не должно.

Для открытия модального окна при отправке в action на верстке указан путь на модальное окно, аналогично href в пункте выше. Метод по умолчанию - GET. Атрибут data-popup-message="#id-модального-окна". Атрибут data-ajax отвечает за работу отправки формы. Если это не нужно - все выше можно убрать, функционал подвязан между собой. Для показа модального окна в ответе ожидается html-разметка, в противном случае будет ошибка.

В случае ошибки отобразится модальное окно с id="status-template".

Если нужно принудительно открыть окно - есть функция window.popup.forceOpen('./popup-callback.html' - ссылка на файл с модальным окном, '#callback-popup' - id модального окна)

### Работа с селектами

Для перерисовки селекта после запроса используется специальная функция - new window.select.constructor(null, `${selector} select`); В функцию передается 2 аргумента, где первый - null, второй - это селектор на select. По умолчанию он стоит от родителя, по желанию можно указать на сам тег select или класс/id на нем

Для отслеживания изменений селекта используется кастомноое событие. Отследить его можно таким образом: document.addEventListener("selectCallback", () => {})

Для отправки селекта при изменении на тег select нужно добавить атрибут data-submit

### Слайдеры

Для отрисовки слайдеров после перерисовки DOM нужно вызвать функцию window.initSliders();

### Карты

Как правило используется Яндекс карта. В зависимости от задачи она будет вставлена либо iframe, либо же на странице будет вставлена такая размета:

<div class="map"  id="map" data-key="Ваш API ключ" data-center="[55.779324, 37.581623]" data-icon="Ссылка на иконку" ></div>
Данные в примере условные и подходят только для одной метки. Тег script с ссылкой на api-maps будет создан автоматически, туда же подставится api-ключ.

### Галерея

В качестве основной библиотеки для галереи используется fslightbox. Для её показа используется ссылка, где href='ссылка-на-картинку' и data-fslightbox="group". Атрибут data-fslightbox группирует галереи, а так же работает в качестве селектора для JavaScript

### Чанки

В теге head присутвует var BASE_URL = './'. Он нужен для корректной работы чанков, если меняются пути. Предаварительно - нужно запросить смену путей на baseUrl у фронта
